Authentication operations

Available endpoints

POST /api/auth/register Register a new user. POST /api/auth/login Login with email and password. POST /api/auth/logout Logout current user. GET /api/auth/me Get current user profile. POST /api/auth/send-otp Send OTP code. POST /api/auth/verify-otp Verify OTP and create a session.

Usage notes

Operational guidance

Online deployment This docs route is served by the same Node app as Mission Control, so it works on the public `sslip.io` deploy without a separate Swagger container.
Hash-safe anchors Links like `#/Authentication/post_api_auth_logout` resolve on this page, matching the original Swagger-style fragment format.
Reference-only surface This page documents the contract. It does not execute logout requests or expose a separate auth UI.
